St. Olaf Security Training

This is a web application by Stone Security, a St. Olaf Capstone project by senior Computer Science majors; Samir Bekhechi '17, Jack Hamby '17, and John Stone '17. The application is intended to provide students with knowledge of current cyber security issues as well as methods to create safer code to prevent malicious activity. Available only around St. Olaf campus the app does not put any college systems at risk and is solely for the purpose of education. We are not responsible for any misuse or unwanted distribution of this application.
